# 🎓 LMS Management System (Java Mini Project)

## 📌 Project Description
This is a Java-based Learning Management System (LMS) developed using JavaFX and MySQL.  
The system allows administrators, lecturers, and students to manage academic activities.

---

## 🚀 Features

### 👤 Admin
- Manage users (Students, Lecturers, Technical Officers)
- Add / Delete courses
- Post and manage notices
- Manage timetable
- View medical records

### 🎓 Lecturer
- View assigned courses
- Manage student marks
- View timetable

### 👨‍🎓 Student
- View courses
- View notices
- View timetable
- Submit medicals

---

## 🛠 Technologies Used
- Java (OOP Concepts)
- JavaFX (GUI)
- MySQL (Database)
- Maven

---

## 🧠 OOP Concepts Used
- Encapsulation
- Inheritance (User → Student, Lecturer, Admin)
- Polymorphism
- Abstraction (DAO layer)
